# Rejection Simulator

Tired of getting rejected by humans and ai systems? `Get Rejected by javascript.`

For when your rejection needs aren't met by normal humans.

## Motivation

In all seriousness I built this for 3 reasons

- I heard that if you remove 1-2 vowels from a verb its a startup name. And I wanted to test that. It works. Hilariously Well.
- Rimworld (a video game) has an art description generator. I wanted to see if I could make something similar. Rimworld art and rejection letters both have this uncanny templated feel so this would be a good place to try it.
- Exposure therapy practice
